# Presentationsteknik — Pedagogiska presentationer för klassrummet

## Grundprinciper

Klassrumspresentationer skiljer sig från konferenspresentationer. De ska inte bara informera — de ska aktivera tänkande och stödja lärande.

### Regler

1. **Max 3 nyckelpunkter per slide** — Mer än tre och eleverna slutar bearbeta
2. **Frågor som slide-titlar** — "Varför startade första världskriget?" istället för "Orsaker till första världskriget". Frågor aktiverar tänkande
3. **Diskussionspaus var 3-4:e slide** — Bädd in en "Diskutera med din granne"-slide regelbundet
4. **Visuellt före text** — Föreslå bildtyp (karta, diagram, fotografi, tidslinje) före textpunkter
5. **Progressiv avslöjning** — Visa inte allt på en gång. Bygg upp resonemang stegvis

## Presentationsflöde (15-20 minuter)

### 1. Öppning (1-2 slides)
- **Hook**: En bild, ett citat, en provocerande fråga eller ett oväntat faktum
- Syfte: Fånga uppmärksamhet och koppla till lektionens kärnfråga
- Exempel: En bild av en historisk händelse med frågan "Vad tror ni händer här?"

### 2. Innehållsblock (3-4 slides per block, max 2-3 block)
Varje block:
- Inledande fråga (slide-titel)
- 2-3 slides med innehåll (max 3 punkter per slide)
- Avslutande diskussions-/reflektionsslide

### 3. Sammanfattning (1 slide)
- Återkoppla till öppningens fråga
- Koppla till den bearbetningsuppgift som följer
- "Nu ska ni..."

## Slide-typer

### Innehållsslide
```
## [Fråga som titel]

- Nyckelpunkt 1
- Nyckelpunkt 2
- Nyckelpunkt 3

> Talarnoter: [Vad läraren säger, förklarar, exemplifierar]
> Visuellt: [Förslag på bild/karta/diagram]
```

### Diskussionsslide
```
## Diskutera (2 min)

[En specifik fråga kopplad till föregående innehåll]

Diskutera med din granne. Var beredd att dela.

> Talarnoter: [Vad läraren lyssnar efter, möjliga uppföljningsfrågor]
```

### Källslide (historia/samhällskunskap)
```
## [Kort rubrik]

[Citat eller bild av källan]

Källa: [Avsändare, årtal, kontext]

Fråga: [Källkritisk fråga att diskutera]

> Talarnoter: [Bakgrundsinformation om källan, vad eleverna bör uppmärksamma]
```

### Jämförelseslide
```
## [Fråga: Vad skiljer X från Y?]

| [X] | [Y] |
|-----|-----|
| Punkt 1 | Punkt 1 |
| Punkt 2 | Punkt 2 |
| Punkt 3 | Punkt 3 |

> Talarnoter: [Vilka likheter/skillnader som är viktiga att lyfta]
```

## Output-format

Generera presentationsinnehåll i Markdown. Läraren överför sedan till valfritt verktyg (PowerPoint, Google Slides, Keynote).

```markdown
# Presentation: [Lektionens titel]

**Lektion:** [N] av [totalt antal]
**Längd:** ca [X] minuter
**Syfte:** [Vad presentationen ska åstadkomma]

---

## Slide 1: [Titel/Fråga]

[Innehåll — punkter, citat, eller instruktion]

> **Talarnoter:** [Vad läraren säger och gör]
> **Visuellt:** [Förslag på bild, karta, diagram eller annat visuellt stöd]
> **Tid:** ca [X] min

---

## Slide 2: [Titel/Fråga]

[Innehåll]

> **Talarnoter:** [...]
> **Visuellt:** [...]
> **Tid:** ca [X] min

---

[Fortsätter för alla slides]
```

## Kvalitetskontroll

Innan presentationen levereras, kontrollera:

- [ ] Har varje slide max 3 nyckelpunkter?
- [ ] Används frågor som titlar (inte bara påståenden)?
- [ ] Finns diskussionspauser minst var 3-4:e slide?
- [ ] Finns talarnoter för varje slide?
- [ ] Finns visuella förslag (inte bara text)?
- [ ] Kopplar presentationen till den efterföljande elevaktiviteten?
- [ ] Matchar innehållet lektionens lärandemål?
- [ ] Är total presentationstid max 20 minuter?
